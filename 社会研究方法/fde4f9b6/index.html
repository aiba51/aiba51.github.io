社会研究方法的优缺点

发表于 2022-05-14 分类于 社会研究方法 阅读次数： 本文字数： 2.5k 阅读时长 ≈ 2 分钟

定量方法

优点：

  1. 能够快速地进行大规模的社会调查。

  2. 通过运用统计和数学分析提高了社会研究的标准化和精确化程度。

  3. 讲求严密的逻辑推理，结论更准确、更科学。

  4. 提高理论的抽象化和概括性程度，能够对社会现象之间的因果关系进行精确的分析。

缺点：

  1. 获得的数据较少、比较表面，很难获得深入、广泛的信息。

  2. 很难客观、准确地确立两个变量之间的因果关系。

  3. 无法得出普遍的经验概括，也无法依赖数量分析。

定性方法

优点：

能够深入理解社会现象本质。

缺点：

  1. 获得的结论难以在较大范围中推论到总体。

  2. 研究成果很难以数学的语言加以精确的描述。

  3. 耗费更长时间。

  4. 标准化程度不高，难以形成大规模社会研究行动。

横向研究

优点：

  1. 调查面广，多数采用统计调查的方式。

  2. 资料的格式比较统一，标准化程度较高。

  3. 资料来源于同一时间，未受到时间变化的影响。

  4. 能够用于描述和解释社会变迁。

缺点：

在进行关于社会变迁的研究中，横向研究的资料只能对变迁进行间接的推测，而不能直接进行论证。

纵向研究

优点：

  1. 能够了解社会现象的发展过程。

  2. 能比较不同时期的变化。

  3. 由于各种变量的时间顺序清楚，纵向研究比横向研究更容易做出逻辑上的因果判断。

缺点：

比横向研究需要更多的时间和经费。

问卷调查

优点：

  1. 面向众多的调查对象收集信息。

  2. 节省资源，包括节省人力、物力、财力和时间。

  3. 具有较好的隐蔽性，减轻研究对象的思想顾虑，有助于获得相对真实的研究信息。

  4. 获得的信息更标准化和规范化。

  5. 获得的信息便于进行定量处理和分析。

缺点：

  1. 通常预设。

  2. 通常以研究对象能够一致地理解问卷内容为前提。

  3. 假定调查对象愿意回答研究者的问题，这一假定是个存在缺陷的假定。

  4. 假定调查对象对于问卷的回答不受调查情境的影响，或者认为可以采用一些技术手段进行控制。

  5. 假定研究对象的回答是真实的。

  6. 假定通过调查数据的分析，能够发现社会现象之间的关系，揭示经验世界的规律性。

  7. 仅仅依据问卷调查所获得的信息，只能得出关于社会现象部分的、表层的、静态的认识。

访谈法

优点：

  1. 信息量大。

  2. 灵活性高。访谈研究的一个主要优点是它的灵活性。

  3. 适用范围广。

  4. 控制性强。

  5. 可以与其它方法相结合获取更加丰富的信息。

缺点：

  1. 开放式的访谈标准不一，难以进行定量研究。

  2. 成本高。

  3. 访谈通常时间长。

  4. 隐蔽性差。

  5. 受访对象周围的环境的影响大。

小组访谈法

优点：

  1. 了解情况快，效率高。

  2. 通过被调查者的相互交流获得更多的信息。

  3. 有利于把调查与研究结合起来，把认识问题与探索解决问题的办法结合起来。

缺点：

  1. 易受被访者心理因素和环境因素的影响。

  2. 保密性差。

  3. 比较费时。

观察法

优点：

  1. 实时地观察到社会现象或行为的发生。

  2. 能够得到不能直接进行访问调查或不便于进行访问调查对象的资料。

  3. 通过特殊的设备或特殊的方式，观察者可以在不干扰被观察者的情况下得到真实的资料。

缺点：

  1. 真正碰到要观察的事件是可遇不可求的。

  2. 人类社会中有许多现象是不适宜或不可能直接观察的，特别是涉及人的隐私行为，如夫妻间的吵架或亲密行为。

  3. 研究事物或现象在长时间内发展变化的历程或趋势有局限性。

  4. 意识到观察者在现场时，得到的资料可能会在一定程度上被扭曲。

实验室观察

优点：

  1. 具有科学研究所要求的精确、客观和可重复验证。

  2. 可以用来研究和发现事物之间的因果关系。

  3. 可以用来收集研究对象的自然发生的行为的最真实、完整、全面的资料。

缺点：

  1. 很多社会现象和人类行为，如集群行为或隐私行为，不可能操纵在实验室中发生。

  2. 实验室中观察到的原则不能够直接推论到实验室以外去解释现实生活的社会现象。

实地观察

优点：

  1. 能够接近生活的真实性。

  2. 能够对社会现象进行深入的考察。

缺点：

  1. 常常无法观察到与研究有关的行为和事件。

  2. 通常是比较个人化的，容易受研究者个人倾向的影响。

  3. 研究结果的可重复验证性较差。

参与观察

优点：受研究者先入之见的影响较小，能够获得研究对象最丰富、最真实的资料 。

缺点：很大程度上依赖于观察者的敏感性、领悟能力和技巧，观察者的主观作用影响很大，研究很难重复，获得的资料缺乏必要的信度。

非参与观察

优点：保证了研究结果的客观性、真实性。

缺点：对研究对象的活动缺乏有效的参与，只能获得表面的资料 。

结构式观察

优点：

可对观察的结果做标准化的记录，可对观察的结果做严格的数理推理，并用来验证研究假设。

缺点：

  1. 要在观察之前确定观察的范畴和标准化的记录工具是一件十分困难的事情，需要事先做大量的探索性研究。

  2. 过于标准化的观察范畴和记录工具将会降低观察的灵活性，超出观察规定和记录标准的信息将不会被收集，减少了所获得资料的丰富性和对社会现实反应的真实性，这反映了结构式观察的局限性。

非结构式观察

优点：

没有严格界定的观察项目和标准化记录方式；优点在于它的弹性，观察者在观察过程中可以表现出足够的灵活性所收集的资料不会受到事先规定的观察项目和记录标准的干扰和限制，能更深入、全面、客观地收集真实反映事物的本来面目的信息。

缺点：

  1. 没有严格界定的观察项目和标准化记录方式；缺点和局限性在于观察过程、观察内容、观察记录会因观察者的不同而不同，所收集的资料带有较多的观察者个人色彩，观察结果缺乏必要的信度，难以重复验证。

  2. 所获得的资料没有统一的形式，通常只能进行定性分析。

直接观察

优点：所得资料的真实性、生动性、客观性和丰富性。

缺点：影响被观察者的行为表现和社会现象的正常进行，影响获得资料及得出结论的真实性和客观性。

间接观察

优点：被观察者的行为不会受到观察的影响，可以收集到反映人的真实行为的资料。

缺点：由于没有观察到行为的发生，所收集到的资料的效度不能保证，很难判断行为和标志或痕迹之间的关系是否具有普遍性。

实验法

优点：

  1. 可检验因果关系。

  2. 较为经济。

  3. 易于重复。

  4. 控制能力较强。

缺点：

  1. 人为干涉。

  2. 样本选取的缺陷。

  3. 实验人员的影响。

  4. 道德和法律的限制。

文献研究法

优点：

  1. 可研究那些不可能接近的研究对象。

  2. 具有非介入性和无反应性的优点。

  3. 适于作纵贯研究和趋势研究。

  4. 费用较低。

缺点：

  1. 文献内容无法控制。

  2. 缺乏统一格式，资料间难以对比。

  3. 编码困难，难以数量化。

  4. 研究所需要的部分文献不易获得。

二手资料分析

优点：

  1. 省钱、省时、省力。

  2. 比较容易做对比研究和趋势研究。

缺点：

  1. 效度难以保证。

  2. 原始资料中包含一些错误和偏差。
